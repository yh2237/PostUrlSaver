# PostUrlSaver

Twitter（現X）のTLからPostのURLを簡単にコピー、保存するためのChrome拡張機能

## 機能

-   Xのタイムライン上の各ポストに保存ボタンを追加します。
-   保存ボタンを押すと投稿のURLを保存したり、クリップボードにコピーしたりできます。
-   ポップアップ画面で、保存されたURLを一覧表示し、以下の操作が可能です。
    -   保存されたURLの個別削除
    -   保存されたURLの全削除
    -   保存されたURLリストのJSONファイルとしてのダウンロード
    -   その他設定
-   ポップアップ画面の下部にある設定セクションで、以下の動作を個別に有効/無効にできます。
    -   保存ボタンクリック時にURLをストレージに保存するかどうか
    -   保存ボタンクリック時にURLをクリップボードにコピーするかどうか
-   自動保存を設定で有効にし、対象アカウントを指定することで、そのアカウントのポストが表示された際に自動的に保存ボタンがクリックされるようにできます。

## インストール方法

この拡張機能はChromeウェブストアには公開されていません。以下の手順で手動でインストールしてください。

1.  [Download ZIP]() 又は [リリース](https://github.com/yh2237/PostUrlSaver/releases/latest) から最新のバージョンをダウンロードします。
2.  Chromeを開き、アドレスバーに `chrome://extensions` を入力して拡張機能管理ページにアクセスします。
3.  ページの右上にある「デベロッパーモード」をオンにします。
4.  「パッケージ化されていない拡張機能を読み込む」をクリックして、解凍した `.zip` の中の `manifest.json` があるディレクトリを選択して読み込む。又は `.crx` ファイルの場合ドラッグ&ドロップで読み込んでください。

**リリースに置いてある `.crx` ファイルからだと警告が出る場合があるのでDownload ZIPからのインストールをお勧めします。**

## 使用方法

### Xのタイムライン上

-   Xのタイムラインを閲覧すると、各投稿の下部に「アクション」ボタンが表示されます。
-   このボタンをクリックすると、設定に応じて投稿のURLが保存されるか、クリップボードにコピーされます。
-   自動アクションが有効な場合、指定したアカウントの投稿は自動的に処理されます。

### ポップアップ画面

-   Chromeのツールバーにある「PostUrlSaver」のアイコンをクリックすると、ポップアップ画面が開きます。
-   **保存されたURL**: 保存済みのURLが一覧表示されます。各URLの横にある「削除」ボタンで個別に削除できます。
-   **JSONをダウンロード**: 保存されているすべてのURLをJSONファイルとしてダウンロードします。
-   **すべてクリア**: 保存されているすべてのURLを削除します。
-   **設定**:
    *   「URLをストレージに保存する」: オンにすると、ボタンクリック時にURLが拡張機能のストレージに保存されます。
    *   「URLをクリップボードにコピーする」: オンにすると、ボタンクリック時にURLがクリップボードにコピーされます。
    *   「特定アカウントのポストで自動実行する」: オンにすると、指定したアカウントの投稿が自動的に処理されます。
    *   「対象アカウント（@は含めず、カンマ区切り）」: 自動実行の対象となるアカウントのユーザー名をカンマ区切りで入力します（例: `elonmusk,jack`）。

## 注意事項

-   この拡張機能はX（旧Twitter）のウェブサイトのDOM構造に依存しています。Xの仕様変更により、一部機能が動作しなくなる可能性があります。
-   自動アクション機能は、Xの利用規約に抵触する可能性があります。ご自身の責任においてご利用ください。
